    Preguntas frecuentes

    ¿Biblio gestiona cuotas e inventario de libros? +

    Sí, además de préstamos Biblio puede gestionar cuotas e inventario de libros. Son funciones opcionales: cada biblioteca decide si las usa o no.

    ¿Necesito conexión a internet para usarlo? +

    No. Biblio es un sistema liviano que funciona sin conexión a internet.

    ¿Qué requisitos técnicos tiene? +

    Es liviano y funciona incluso en Windows 7. Por ahora solo está disponible para computadoras, no funciona en tablets ni celulares.

    ¿Es compatible con otros sistemas como Koha? +

    Sí, es compatible con sistemas como Koha. El inventario puede migrarse de uno a otro.
